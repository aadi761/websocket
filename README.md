# 🚀 WebSocket Real-Time Chat App

A full-stack real-time chat application built using WebSockets, enabling instant bidirectional communication between clients and server.

---

## 🧠 Overview

This project demonstrates how real-time communication works using WebSockets.
Users can send and receive messages instantly without refreshing the page.

---

## 🛠 Tech Stack

### Frontend

* React (Vite)
* CSS

### Backend

* Spring Boot
* WebSocket (STOMP protocol)

### Other

* REST APIs
* Java

---

## ⚡ Features

* 🔴 Real-time messaging (no refresh needed)
* 🔄 Bidirectional communication using WebSockets
* 📡 STOMP protocol-based messaging
* 🧩 Modular full-stack architecture
* 🎯 Clean UI with responsive behavior

---

## 📂 Project Structure

```
fs10/
│── Demo_WebSocket/        # Spring Boot backend
│── frontend/websocket/    # React frontend
```

---

## ▶️ How to Run

### 1. Clone the repo

```
git clone https://github.com/your-username/your-repo.git
```

---

### 2. Run Backend

```
cd Demo_WebSocket
./mvnw spring-boot:run
```

---

### 3. Run Frontend

```
cd frontend/websocket
npm install
npm run dev
```
